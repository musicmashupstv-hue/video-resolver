🎬 Video Resolver API

This service converts share links (Google Drive, Starchive, etc.) into direct video URLs for use with native <video> players.

Endpoint

/.netlify/functions/resolve?url=ENCODED_URL

Example

https://video-resolver.netlify.app/.netlify/functions/resolve?url=https://drive.google.com/file/d/ABC123/view

Response

{
  "url": "https://drive.google.com/uc?export=download&id=ABC123",
  "success": true
}

If the resolver cannot find a direct video URL, it returns success: false and the frontend should open the original link in a new tab.
